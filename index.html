How to Set Up Add-ons

  1. Open Kodi and go to Settings (gear icon).
  2. Select System → enable Unknown sources.
  3. Return to Settings → open File Manager.
  4. Add source → enter the repository URL → name it.
  5. Go back to the main menu → Add-ons → Install from zip file.
  6. Choose your source → install the repo zip file.
  7. Then select Install from repository → pick your add-ons.
