Kodi

Kodi is an open-source media center app that plays videos, music, and more through add-ons.

How to Setup the Lab Add on

  1. Open Kodi and go to Settings (gear icon).
  2. Select System → enable Unknown sources.
  3. Return to Settings → open File Manager.
  4. Add source → (https://tinyurl.com/TheL48.
  5. Click ok→ Name it L48 and click ok → Click back 1x and select add on's.
  6. Install from zip file → Select The Lab Repo
  7. Select only repo available You will see an install message upper right corner → Now install from repository >Select The Lab>Select video add ons>Install Dream Machine & Dream TV for Live TV, 24/7 channels and PPV.

Popular Kodi Sources

    How to Install Jew Repo (Scrubs V2)

    1. Go to Settings → System → enable Unknown sources.
    2. Click back one time and open File Manager.
    3. Scroll down to Add source.
    4. Type in URL: https://jewbmx.github.io
    5. Click OK → name it Jew → click OK again.
    6. You will see it saved on the right.
    7. Click back → go to Add-ons.
    8. Select Install from zip file.
    9. Choose Jew → select the repo file → wait for install message.
    10. Click Install from repository → open Jew Repo.
    11. Go to Video add-ons → find Scrubs V2 → click Install.
