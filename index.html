Kodi Pro Tips ✅

✅ 5. TWEAK SETTINGS FOR PERFORMANCE

  • Settings → Player → Videos → Set Allow hardware acceleration ON
  • Settings → Services → UPnP / Control → Turn OFF anything you don’t use

🧠 Less clutter = faster, smoother Kodi

✅ 6. SET UP AUTO-CLEANING

  • If using a wizard (like cMaN or EzzerMacs), go to:
  • Maintenance → Auto-Clean on Startup → Enable

🧻 Clears cache, old packages, thumbnails — keeps Kodi snappy

✅ 7. KEEP AN ADD-ON JOURNAL (Optional but Boss Move)

  • 📝 Jot down what you installed, from where, and when.
  • 💡 Helps when something breaks or you want to reinstall your vault on a fresh Kodi

🧠 Bonus: Stremio Privacy Settings

  1. Open Stremio → tap Settings (gear icon)
  2. Go to General or Streaming Settings
  3. Enable:
    • ✅ Use external player (optional)
    • ✅ VPN recommended (for anonymity)
  4. Disable telemetry if you want more privacy
